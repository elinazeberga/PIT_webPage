Installation:

npm init -y
npm install

manual installation, if no package.json:
npm install express mongoose bcryptjs jsonwebtoken dotenv nodemon


Test installation:

manual installation, if no package.json:
npm install jest supertest --save-dev


In package.json overwrite script tag with
{
  "scripts": {
    "test": "jest"
  }
}


.env file creation:

JWT_SECRET=paskaties whatsappā
MONGODB_URI=paskaties whatsappā


Running: 
(backend and static frontend)

npm run dev
open this -> http://localhost:3000/


Run tests, close app:

npm test

Better to run single test not all at the time DB response is too slow for that:
example:
npx jest tests/routes/bookings.test.js
